"Ao longo de muitos estudos tenho realizado alguns projetos, e para conhecer um pouco da minha capacidade, estarei falando um pouco dos projetos recentes. Todos feitos depois de uma revisão de uma forma pratica. "


Projeto Unes

Unes – criação de site, utilizando listas e formatação de texto, meu primeiro projeto onde aprendi a fazer cabeçalhos, Parágrafos, Lista ordenada e não ordenadas, imagens, links, formulários, caracteres especiais e como todo primeiro site todo o projeto feito tudo em html sem utilização de css.


Projeto Ana Bella

Ana Bella – site criado com css. Foi meu primeiro site em css onde aprendi sobre Inline css, Seletores css, Classes e ID’s, Tags Div & Span, Bordas, Fontes & Cores, Tamanhos de textos, Estilos de textos, Cor e imagem de fundo, CSS Externo e assim construímos o site Ana Bella.


Projeto TecBlog

TecBlog – Além de ter utilizado todas as ferramentas assim como o site Ana Bella que foi feito todo em estilo css, o TecBlog foi um pouco além com Modelo de Caixa, Elementos Flutuantes, Elementos Inline, Block e Inline-Block, Posicionamento estático e relativo, Absoluto e fixo, Sobrepondo elementos, Formatando Links.


Projeto Noticia Cidade

Site Noticia Cidade – criado em Css um pouco mais avançado, utilizando mais sobre seletores, Herança e Especificidade, Barra de navegação vertical, Marcação de Pagina Atual, Navegação de Pagina horizontal, Navegação com Abas, Layout de largura fixa, Layout líquido, imagens liquido, Coluna falsa, Estilizando tabelas e formulários, Arredondamento de imagens e muito mais e assim começou os conhecimentos a fazer sentido na criação de sites.


Projeto Chalé Hotel

Projeto Chalé hotel, um projeto com um layout muito vibrante com cores e imagens atrativas. Utilizando seletores, especificidade, layout liquido entre outros aprendizado um pouco a mais do site noticia cidades.


Projeto Parallax

Parallax - site criado com layout bastante divertido parallax, bastante utilizado na internet e com todos os conhecimentos já utilizados em sites anteriores. Os efeitos são incríveis.


Projeto Museu Nacional

Museu Nacional – Um site que já vai tornando um pouco mais profissional com estudos baseados em Normalize css, Box Sizing & Cantos arredondados, Degradês e Transparências, Sombras, Animações, Transições, Novas tags html 5 Header, Nav e Footer, Article, Section, Aside e time, compatibilidade html 5, novas tags de input, flexbox, container, itens e muito mais.


Projeto Finans

Finans – site baseado na responsividade do Bootstrap, com aprendizagem em Formatações de textos, Alinhamento de textos, Elementos flutuantes, Cores & Backgrounds, Margin & Padding, Tamanhos & Bordas, Media Queries, Botões, Barra de navegação, Listas, Formularios, Input group, Alertas & Barra de progresso, Tabelas & Paginação, Cards, Jumbotron & Alinhamento de imagens, Ícones, Sistema de grid, Configurações do grid, Classes do Flexbox, Margin & Wrap, Projeto padão entre outros para deixar o site bastante responsivo.


Projeto Spotify

Spotfy – um site incrível com a responsividade do Bootstrap, com todas as aprendizagem antes falado no Finans, em Formatações de textos, Alinhamento de textos, Elementos flutuantes, Cores & Backgrounds, Margin & Padding, Tamanhos & Bordas, Media Queries, Botões, Barra de navegação, Listas, Formularios, Input group, Alertas & Barra de progresso, Tabelas & Paginação, Cards, Jumbotron & Alinhamento de imagens, Ícones, Sistema de grid, Configurações do grid, Classes do Flexbox, Margin & Wrap, Projeto padão entre outros para deixar o site bastante responsivo e muito profissional.
